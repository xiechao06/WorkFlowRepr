types of order¶

class work_flow_repr.Node(name, description, target)¶

this class represent an object (draw in dashed line) in work flow DAG.

Parameters:
  • name – name of the node
  • description – description of the node
  • target – target of the node
children_groups¶

return a groups of children

Returns:a list of children groups, each group should have 2 attributes:
  • name - name of the group
  • items - children objects
events¶
Return type:a list of work_flow_repr.Event
json¶

represent a tree root from self

Return type:DictType
class work_flow_repr.Event(datetime, name, actor, description='', by='')¶

represent an event in work flow DAG

Parameters:
  • datetime (datetime.datetime) – datetime of the event
  • name – name of the event
  • actor – the actor of the event
  • description – description of the event
By :

if the event is the birth event of a node, and the node is generated by another node’s event (say branch event), then by should be the name of the branch event
json¶

represent the event as dict

Return type:DictType
